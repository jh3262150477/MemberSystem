# 会员管理系统 (Member Management System)

## 📋 项目简介

一个基于 C++ 开发的会员管理系统，提供完整的会员信息管理、积分管理、消费记录管理和系统设置功能。

### ✨ 主要功能

- **会员信息管理**：添加、查询、修改、删除会员信息
- **积分管理**：自动积分计算、积分兑换、积分历史查询
- **消费记录管理**：消费记录、统计、查询消费明细
- **等级系统**：自动等级升级、折扣优惠
- **数据持久化**：支持数据保存和加载
- **系统设置**：积分规则设置、会员等级预测

## 🚀 快速开始

### 环境要求

- **操作系统**：Windows 10/11
- **编译器**：MinGW-w64 (GCC 13.1.0 或更高版本)
- **构建工具**：CMake 3.10 或更高版本
- **IDE**：VS Code (推荐)

### 安装步骤

1. **克隆项目**
   ```bash
   git clone <repository-url>
   cd MemberSystem
   ```

2. **创建构建目录**
   ```bash
   mkdir build
   cd build
   ```

3. **配置 CMake**
   ```bash
   cmake .. -G "MinGW Makefiles"
   ```

4. **编译项目**
   ```bash
   mingw32-make
   ```

5. **运行程序**
   ```bash
   cd bin
   MemberSystem.exe
   ```

项目已包含完整的 VS Code 配置：

1. **使用 VS Code 任务**：
   - 按 `Ctrl+Shift+P`
   - 选择 "Tasks: Run Task"
   - 选择 "C/C++: g++.exe 生成活动文件"

2. **使用快捷键**：
   - `Ctrl+Shift+B`：运行默认构建任务

3. **调试运行**：
   - `F5`：启动调试

### 会员等级系统

| 等级 | 年度消费要求 | 折扣优惠 | 积分规则 |
|------|-------------|----------|----------|
| 普通会员 | < 5000元 | 无折扣 | 1元=1积分 |
| 白银会员 | ≥ 5000元 | 95折 | 1元=1积分 |
| 黄金会员 | ≥ 10000元 | 9折 | 1元=1积分 |
| 钻石会员 | ≥ 20000元 | 8折 | 1元=1积分 |

### 数据格式

会员数据以 CSV 格式保存，包含以下字段：
- 会员ID
- 姓名
- 电话
- 生日
- 总消费金额
- 当前积分
- 积分规则
- 年度消费
- 会员等级
- 上次消费年份

## 🐛 故障排除

### 常见问题

1. **编译错误**：
   - 确保安装了 MinGW-w64
   - 检查 CMake 版本是否满足要求
   - 确保所有源文件都在项目中

2. **中文显示问题**：
   - 使用外部控制台运行
   - 设置控制台编码为 UTF-8

3. **链接错误**：
   - 确保使用 CMake 构建而不是直接编译
   - 检查所有源文件是否包含在构建中

### 调试技巧

- 使用 `F5` 启动调试
- 在 VS Code 中设置断点
- 查看控制台输出和错误信息
